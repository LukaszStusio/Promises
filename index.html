Making a pizza excercise

You can't make a pizza instantly. You have to bake it, put toppings, etc. That takes time.

When you order a pizza you get an order-ticket. You can't eat it - it's just a promise that they do it for you.

Check console logs of course.

Excercise features

  • Chaining with .then()
  • Running functions sequentially vs concurrently
  • MEGA Promise - a static method that takes an array of promises as argument
  • Promise.race() - waits for the first element in the array.

Reject() handling

Reject a makePizza() function if people try to put a pineapple in it.

  • catch() - handles errors from reject().
  • It stops the chain of .then() if evoked, so the rest of it can't run even it's correct.
  • Promise.allSettled() doesn't brake the chain.
